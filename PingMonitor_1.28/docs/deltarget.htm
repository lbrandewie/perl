deltarget

The deltarget program deletes an entry from targets.txt in the system folder. It can be used two ways. In the compact form, only the display name or id of a target is specified. In the verbose form, you can use the -n argument to specify the display name or the -i argument to specify an id.

In other words, either:

./deltarget [name or id of target]

Or,

./deltarget -n [name]

Or,

./deltarget -i [id]

Providing the target can be found, it will be deleted. In the compact form, if the argument looks like an id (a t followd only by numbers) it will be assumed to be an id, otherwise it will be assumed to be a name.
