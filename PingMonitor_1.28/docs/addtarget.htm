addtarget

The addtarget program adds an entry to targets.txt in the system folder. There are two syntaxes, compact:

./addtarget [display name] [target name/address] [yellow threshold] [group assignment]

Or, the verbose:

./addtarget -n [display name] -t [target name/address] -y [yellow threshold] -g [group assignment]

If the display name or group name contain spaces, be sure to use quotes around them. Yellow threshold is measured in milliseconds. The group you are assigning to must already exist.
