edittarget

The edittarget program allows you to edit entries in the targets.txt file in the system folder. The first argument is the id of a target to be edited. The remainder of the arguments are pairs of options and values which describe what you want to do to the entry.

./edittarget [id] -n [new name] -t [new target] -y [new yellow threshold] -g [new group]

If you specify a new group, the group must already exist. The yellow threshold must be at least 0.1 ms. Be sure to enclose any new name in quotes if it contains spaces.
