editpage

The editpage program allows you to change the name of a page in pages.txt in the system folder. Pages have no other properties.

The program takes two arguments; the id of a page to be edited, and the new name for that page. Be sure to enclose the new name in quotes if it contains spaces.

In short,

./editpage [id] [new name of page]
